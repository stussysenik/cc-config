Show Claude Code usage statistics with token counts, costs, cache savings, and subscription value.

Usage: `/stats [date-range]`

Examples:
- `/stats` - All time statistics
- `/stats 7d` - Last 7 days
- `/stats 30d` - Last 30 days
- `/stats last-month` - Previous month
- `/stats this-month` - Current month

```bash
python3 ~/Desktop/cc-config/sync-native-logs.py --stats $ARGUMENTS
```
